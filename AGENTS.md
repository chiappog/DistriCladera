# AGENTS.md

Este proyecto utiliza un proceso de **Razonamiento Metacognitivo** para garantizar la calidad y precisión de las soluciones técnicas.

## Reglas Principales

- **Proceso Obligatorio**: Para cualquier tarea compleja, sigue el flujo: DESCOMPONER → RESOLVER → VERIFICAR → SINTETIZAR → REFLEXIONAR.
- **Confianza Explícita**: Indica siempre el nivel de confianza (0.0-1.0) en tus propuestas.
- **Formato de Entrega**: Todas las respuestas deben incluir una "Respuesta Clara", "Nivel de Confianza" y "Advertencias Clave".

## Referencias Externas

Para ver el proceso detallado y las escalas de confianza, consulta:
- @.cursor/rules/metacognitive-reasoning.mdc

## Estructura del Proyecto
- `components/`: Componentes de la UI (React + Tailwind).
- `contexts/`: Gestión de estado global.
- `hooks/`: Lógica reutilizable.
- `views/`: Páginas principales de la aplicación.
- `utils/`: Funciones de utilidad y exportación.
